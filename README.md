## Description

Add friends, post thoughts and react to other people's thoughts in this API framework for a social media platform.

[Walkthrough video](https://youtu.be/WhiW8BIAF-c)

## Installation

Install with:

```md
npm i
```
<br/>

## Usage

Start with:

```md
npm start
```
